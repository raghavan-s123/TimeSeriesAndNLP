# Analysis Description 



# 1.1 Simple Moving Averages (SMA) 



# The first visualization overlays the actual Close price with: 



# 10-day SMA (short-term trend) 
# 30-day SMA (long-term trend) 


# Observations 



# The 10-day SMA closely follows the actual price, reacting quickly to short-term price changes and volatility. 
# The 30-day SMA is smoother and lags behind the price, clearly capturing the long-term upward trend. 
# Crossovers between the 10-day and 30-day SMAs indicate potential buy or sell signals, where: 
# Short-term SMA crossing above the long-term SMA suggests bullish momentum. 
# Short-term SMA crossing below the long-term SMA suggests potential bearish movement. 
 

# 1.2 Simple Exponential Smoothing (SES) 



# The second visualization compares the original Close price with the SES-smoothed series. 



# Observations 



# SES assigns more weight to recent observations, making it more responsive than long-period SMAs. 
# The smoothed curve effectively reduces short-term noise while preserving the overall trend. 
# SES adapts faster to trend changes than the 30-day SMA but remains smoother than the raw price data.
